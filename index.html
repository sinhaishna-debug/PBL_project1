Crop Disease Detector with Weather-Based Early Warning

A hybrid machine learning approach integrating CNN-based crop disease detection with climate-driven risk prediction for early farmer alerts.

Registration ID: 2427030828

Introduction

Crop diseases significantly reduce agricultural productivity due to delayed detection. Farmers often lack technical tools for early diagnosis. Weather conditions such as humidity, rainfall, and temperature accelerate disease spread. This project proposes a hybrid ML-based early warning system combining image analysis and weather data.

Literature Survey

  • Deep CNNs provide high accuracy but require large datasets
  • Weather-based risk prediction enables preventive alerts
  • Most existing mobile tools lack hybrid prediction
  • Hybrid image + climate systems improve reliability
  • PlantVillage dataset is widely used in crop disease research

Problem Statement

Crop diseases and climatic variations cause large-scale agricultural losses. Farmers lack accessible tools for early disease detection and climate-based risk prediction. Existing systems focus on isolated approaches, reducing prediction accuracy.

Objectives

  • Develop CNN-based crop disease classification
  • Build weather-driven disease risk prediction model
  • Integrate both models for early-warning alerts
  • Improve reliability using hybrid prediction
  • Deploy a lightweight demonstration prototype

Proposed Methodology

  1. Collect and clean leaf image datasets
  2. Gather weather parameters (temperature, humidity, rainfall)
  3. Preprocess data using normalization and augmentation
  4. Train CNN for disease detection
  5. Train ML model (RF / XGBoost / LSTM) for climate risk
  6. Integrate models with weighted scoring
  7. Develop dashboard for visualization

Current Project Status

This project is currently in the design and planning phase. Dataset identification, algorithm selection, and system architecture have been completed. Implementation and testing will be carried out in the next phase of PBL.
